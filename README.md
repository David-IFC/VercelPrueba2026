# Demo para Vercel

Proyecto de prueba listo para desplegar en Vercel:

- `index.html`: interfaz principal
- `main.js`: llamada a la API
- `api/health.js`: funcion serverless

## Como subirlo

1. Instala Vercel CLI si no la tienes: `npm i -g vercel`
2. En esta carpeta ejecuta: `vercel`
3. Para produccion: `vercel --prod`

## Desarrollo local

1. Instala dependencias: `npm install`
2. Arranca el entorno local: `npm run dev`
